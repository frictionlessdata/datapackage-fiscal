## Specifications

The Fiscal Data Package Standard contains a core specification and a taxonomy:

<CardGrid>
  <Card title="Fiscal Data Package" icon="open-book">
    Fiscal Data Package is a lightweight and user-oriented format for publishing and
    consuming fiscal data.
  </Card>
  <Card title="Fiscal Data Package - taxonomy" icon="document">
    The Budget Taxonomy is a set of ColumnTypes to be used in the context of a Fiscal Data
    Package to describe spending data of organizations (governments or otherwise)
  </Card>
</CardGrid>
